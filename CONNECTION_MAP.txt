Location: src/utils/skillMatch.js (63 lines)

Function: calcMatchScore(seekerSkills, jobSkills)

Purpose: Calculate % match between:
  - seekerSkills    [{skillId, yearsExp}, ...]
  - jobSkills       [{skillId, requiredLevel, weight}, ...]

Return: 0-100 (percentage)

Algorithm:
  1. Map seeker skills by skillId → yearsExp
  2. For each job skill:
     - If missing: penalty if required
     - If has skill: calculate level gap
     - Gap 0: score = 1.0 (100%)
     - Gap 1: score = 0.6 (60%)
     - Gap 2+: score = 0.2 (20%)
  3. Weighted average of all scores

Used by:
  ✅ JobBrowse.jsx (line ~75)   - Show match % in list
  ✅ JobDetail.jsx (line ~60)   - Show match % prominently
  ✅ JobCard.jsx (line ~12)     - Show match % badge
